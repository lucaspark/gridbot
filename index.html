GridBot
The obedient grid system

DownloadGitHub

GridBot provides a truly easy-to-use, no frills, responsive grid system. Structure is consistent, results predictable AND it's fully able to provide different grid widths between desktop and mobile sizes. GridBot can be as simple as you want; it can also be edited to change breakpoints, grid number and gutter width.

GridBot lives to serve humans. It is simple to set up your page for grids:

<div class="container">
  <div class="group">
    <div class="d6"><!--Content--></div>
    <div class="d6"><!--Content--></div>
  </div>
</div>

.d is for desktop size, .m for mobile size, .o to offset. Feel free to drop the container if you do not want to restrict your content to a max-width.

GridBot serves desktop and mobile devices equally. Observe:

<div class="container">
  <div class="group">
    <div class="d3 m6"><!--Resize browser--></div>
    <div class="d3 m6"><!--to see how--></div>
    <div class="d3 m6"><!--mobile width differs--></div>
    <div class="d3 m6"><!--from desktop!--></div>
  </div>
</div>

A mobile class must be accompanied by a desktop class.

Offsets? Nested grids, you ask? GridBot tires of your skepticism:

<div class="container">
  <div class="group">
    <div class="d4 o2"><!--Offset 2!--></div>
    <div class="d6">
      <div class="group">
        <div class="d5"><!--Nested!--></div>
        <div class="d7"><!--Nested!--></div>
      </div>
    </div>
  </div>
</div>

The CSS file is 1KB minified. Add it to your current project for a ready-made grid system. GridBot came about after messing with many iterations of many different grids. If it doesn't suit your needs as is, feel free to break it apart and use what you want. There is also an SCSS file for easy changes if you work with SASS.
